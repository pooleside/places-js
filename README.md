# _Places You've Been (JavaScript)_

#### By _Leslie Poole_

## Description
A program that allows the user to enter a place they have visited along with additional details: time of year they visited, landmarks and notes.  Each new place will be added to a list of all their places.

## Setup

* Clone this git repository on your computer.
* Open the count-up.html file in your browser(Chrome) to run the program.

## Technologies Used

JavaScript, JQuery, Mocha, Chai, Bootstrap, HTML, and CSS
